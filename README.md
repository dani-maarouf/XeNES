# XeNES

## Input
| NES  | Keyboard |
|------|----------|
|A     | SPACE    |
|B     | LCTRL    |
|Up    | w        |
|Down  | s        |
|Left  | a        |
|Right | d        |
|Start | x        |
|Select| c        |

## Usage

#### Linux:

Run `make` to compile program

Use program by running `./nes ROM`
